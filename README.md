### How to install and run locally?

```
git clone https://github.com/arjunrajput18/speedyBuyWeb.git
cd ElectronicsCentral
npm install
npm start
```

  - Category
  - Ratings
- Wishlist Management
- Cart Management
- Search
- Loading & Alerts
- Authentication
- Sign-up
- Login

- Checkout
